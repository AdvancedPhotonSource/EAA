Thoroughly describe the attached image or images. Focus only on visible evidence, and say when an identification is uncertain.

Organize your description with this structure:

1. High-level overview: In one or two sentences, summarize what the image is. If it contains several panels, blocks, plots, or views, describe their arrangement.
2. Axes and coordinate systems: If the image contains coordinate axes, describe the x and y axis labels, units, scales, tick ranges, and any legends or color bars.
3. Image content and plots: For photographs, microscope images, rendered arrays, and other 2D images, identify the semantically meaningful objects or regions. Describe their appearance, approximate size, and location. Use coordinate values when axes are visible; otherwise use fractions of image width and height. For graphs, describe the curve or data shape, trends, key features, peaks, valleys, outliers, and main takeaways with approximate coordinates and values when visible.
4. Text and markers: Read out visible text labels, annotations, symbols, markers, scale bars, legends, and numeric values.
5. Practical summary: Finish with the most important observations that another agent should know before reasoning about or acting on the image.
